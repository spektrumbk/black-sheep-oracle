__ SCENARIOS

3 ท่าทดสอบตามมาตรฐาน DLT

ครบทุกท่าสอบที่ใช้ในสนามจริงของกรมการขนส่งทางบก

🟩
SCENARIO 01
ขับตรง — ถอยตรง

ทดสอบการควบคุมรถให้ตรง ไม่เฉียวทับเส้น ทั้งเดินหน้าและถอยหลังในเลน 2.5 เมตร ระยะ 20 เมตร

  • ล้อทับเส้น หัก 20 คะแนน
  • วิ่งครบ ไป-กลับ ถึงสำเร็จ
  • ฝึกการควบคุมพวงมาลัย
🟦
SCENARIO 02
หยุดเทียบทางเท้า

จอดรถเทียบขอบทางเท้า โดยต้องจอดห่างขอบไม่เกิน 25 ซม. พร้อมหยุดตรงจุดที่กำหนด

  • ห่างขอบเกิน 25cm หัก 10 คะแนน
  • ปีนขอบทาง หัก 20 คะแนน
  • ต้องชะลอและหยุดในจุดที่กำหนด
🟥
SCENARIO 03
ถอยเข้าซอง

ถอยรถเข้าลานจอดหรือที่จอดรถ โดยไม่ชนเสาและไม่เปลี่ยนเกียร์เกิน 7 ครั้ง

  • ชนเสาหรือขอบ หัก 20 คะแนน
  • เปลี่ยนเกียร์เกิน 7 ครั้ง หัก 10
  • จอดนิ่งในช่องจอดถึงสำเร็จ

__ TECHNOLOGY

⚡ เทคโนโลยีที่ใช้

สร้างด้วยเว็บเทคโนโลยีสมัยใหม่ — ไม่ต้องติดตั้ง ไม่ต้อง Plugin

🎨
Three.js

3D Rendering Engine สำหรับเว็บ · สร้างกราฟิก 3D คุณภาพสูงผ่าน WebGL โดยไม่ต้องใช้ plugin

HTML5 + JavaScript

🚀
GitHub Pages

🏎️
